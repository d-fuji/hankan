## 作業手順

各バッチで:

1. 家（clans.json）に新規家追加
2. 領地（territories.json）に新規藩追加
3. 人物（persons.json）に全藩主追加（親→子の順序厳守）
4. 役職履歴（appointments.json）に全藩主の就任記録追加
5. 石高履歴（kokudaka.json）に各藩2〜3件追加
6. `npm run db:seed` でシード投入検証
7. `npx vitest run` でテスト通過確認
8. コミット
9. `npm run db:prod:seed` で本番反映
